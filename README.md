# XUL-IMAI

**XUL Chain AI Marketing Platform**

基于 IMAI.WORK 构建的 XUL 链版本，集成了区块链技术。

## 🌟 特性

- 🤖 **AI 获客** - 视频号、小红书、抖音、快手自动获客
- 📹 **矩阵运营** - 多平台一键自动发布
- 👥 **AI 员工** - 智能聊天、朋友圈自动互动
- 📱 **AI 手机** - RPA 自动化手机操作
- ⛓️ **区块链集成** - 钱包登录、代币支付、链上身份

## 🔗 XUL 链集成

| 功能 | 说明 |
|------|------|
| 钱包登录 | 钱包签名认证，无需手机号 |
| 代币支付 | XUL/POWER 代币支付服务费 |
| Agent SBT | AI Agent 链上身份认证 |
