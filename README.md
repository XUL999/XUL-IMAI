# XUL-AI

**XUL Chain AI Integration Platform**

白标人工智能集成应用，基于开源生态构建的企业级 AI 服务平台。

## 🌟 特性

- 🤖 **AI 获客** - 视频号、小红书、抖音、快手自动获客
- 📹 **矩阵运营** - 多平台一键自动发布
- 👥 **AI 员工** - 智能聊天、朋友圈自动互动
- 📱 **AI 手机** - RPA 自动化手机操作
- ⛓️ **区块链集成** - 钱包登录、代币支付、链上身份

## 🔗 XUL 链集成

| 功能 | 说明 |
|------|------|
| 钱包登录 | 钱包签名认证，无需手机号 |
